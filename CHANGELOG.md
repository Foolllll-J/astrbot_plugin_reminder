# 📝 更新日志

## v1.4.2 (2026-04-03)
  * 修复 启动/停止提醒等指令未完整支持“名称或序号”的问题
  * 修复 链接指令派发事件使用秒级 `message_id` 导致可能触发的问题
  * 优化 链接提醒相关提示文案，并统一命令注释和说明
  * 清理 已废弃的 `monitor_timeout` 遗留配置项

## v1.4.1 (2026-03-25)
  * 新增 执行轮次参数 `[-数字]`，支持在添加/编辑提醒与任务时设置上限
  * 优化 参数解析流程，统一处理会话、轮次、cron 与内容组合输入
  * 优化 调度模型为“单任务单 job”，在多会话中统一分发执行

## v1.4.0 (2026-03-19)
  * 新增 立即发送提醒（/发送提醒）与立即执行任务（/执行任务）
  * 优化 编辑提醒/任务体验，支持仅修改会话/cron/撤回/内容任意组合
  * 重构 项目结构


## v1.3.0 (2026-03-15)

  * 新增 支持通过引用消息直接设置提醒内容
  * 新增 支持引用语音、视频作为提醒内容

## v1.2.0 (2026-03-10)

  * 支持 提醒消息自动撤回
  * 支持 启停 提醒/任务 指令单次传入多个会话参数
  * 优化 提醒内容中的艾特解析
  * 更改 会话类型前缀标识符

## v1.1.2 (2026-02-21)

  * 修复 删除/查看 任务/提醒时，传入数字参数导致的类型错误

## v1.1.1 (2026-02-21)

  * 支持 提醒和任务中使用 Face (QQ表情) 和 At 组件
  * 修复 指令执行时的上下文丢失问题

## v1.1 (2026-01-17)
  
  * 支持 `#好友号` 指定私聊会话
  * 新增 `编辑提醒`、`编辑任务` 指令
  * 新增 `启动提醒`、`停止提醒`、`启动任务`、`停止任务` 指令
  * 新增 `白名单用户` 配置项

## v1.0
  
  * 新增 指令任务 模块
  * 新增 链接提醒 功能
  * 新增 查看/删除链接 命令

## v0.1
  
  * 实现 定时提醒 功能
  * 支持 使用 cron 表达式创建定时提醒
  * 支持 图文混排消息，完美保持原始顺序
