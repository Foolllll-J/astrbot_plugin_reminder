# 📝 更新日志

## v1.3.0 (2026-03-15)

  * 新增 支持通过引用消息直接设置提醒内容
  * 新增 支持引用语音、视频作为提醒内容

## v1.2.0 (2026-03-10)

  * 支持 提醒消息自动撤回
  * 支持 启停 提醒/任务 指令单次传入多个会话参数
  * 优化 提醒内容中的艾特解析
  * 更改 会话类型前缀标识符

## v1.1.2 (2026-02-21)

  * 修复 删除/查看 任务/提醒时，传入数字参数导致的类型错误

## v1.1.1 (2026-02-21)

  * 支持 提醒和任务中使用 Face (QQ表情) 和 At 组件
  * 修复 指令执行时的上下文丢失问题

## v1.1 (2026-01-17)
  
  * 支持 `#好友号` 指定私聊会话
  * 新增 `编辑提醒`、`编辑任务` 指令
  * 新增 `启动提醒`、`停止提醒`、`启动任务`、`停止任务` 指令
  * 新增 `白名单用户` 配置项

## v1.0
  
  * 新增 指令任务 模块
  * 新增 链接提醒 功能
  * 新增 查看/删除链接 命令

## v0.1
  
  * 实现 定时提醒 功能
  * 支持 使用 cron 表达式创建定时提醒
  * 支持 图文混排消息，完美保持原始顺序
