# 定时提醒插件

AstrBot 定时提醒插件，支持定时发送消息到群组或私聊，支持 cron 表达式和图片消息。

## 功能特性

- ✅ 支持 cron 表达式定时任务
- ✅ 支持发送到群组或私聊
- ✅ 支持文本和图片消息

## 使用方法

⚠️ **所有管理指令仅限Bot管理员使用**

### 添加提醒

```
/添加提醒 <名称> <目标类型> <目标ID> <cron表达式> <消息>
```

- **目标类型**: `群组` 或 `私聊`
- **cron表达式**: 5段格式 (分 时 日 月 周)
- 💡 **发送指令时可以同时附上图片**，提醒会包含文字+图片

**示例**:

```
/添加提醒 早安 群组 123456 0 9 * * * 早上好！
```

每天9点发送"早上好！"到群123456

```
/添加提醒 早安图 群组 123456 0 9 * * * 早上好！
```

同时附上一张图片，每天9点发送文字+图片到群123456

```
/添加提醒 周报 私聊 987654 0 18 * * 5 本周工作总结
```

每周五18点发送到私聊987654

### Cron 表达式说明

格式: `分 时 日 月 周`

- **分**: 0-59
- **时**: 0-23
- **日**: 1-31
- **月**: 1-12
- **周**: 0-6 (0=周日)
- **\***: 任意值
- **\*/n**: 每n个单位

**常用示例**:

- `0 9 * * *` - 每天9点
- `0 */2 * * *` - 每2小时
- `30 18 * * 1-5` - 工作日18:30
- `0 0 1 * *` - 每月1号0点
